Media, images and other pretty things

Media is another concern in modern design. Videos should be responsively sized for both desktop and mobile. More commonly, we might simply include an embed, and pass the hosting costs onto someone else, like Google.

A lot of the same advice goes for images, with an added emphasis on size. Most modern browsers support internet optimized image formats like WebP and avif, which can be substantially smaller than conventional formats. Supporting multiple versions of a single image can help ensure that when possible, the most efficient image is served while ensuring compatibility.
