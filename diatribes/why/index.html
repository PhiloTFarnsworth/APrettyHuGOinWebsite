Why Another Heckin' Website?

This site seeks to model a text heavy informative website (like a manual or textbook) while also utilizing many of the current available HTML elements. We’ll also create a small but sensible set of styling instructions in CSS and a couple of handy but optional JS features.

Learning web design?

Get ready to be schooled!

By doing so across both Single Page Application (SPA) and static experiences, we ensure our content is accessible, both to users who may have conditions that impair their ability to browse a website, as well as to users who have a revulsion to running Javascript.
